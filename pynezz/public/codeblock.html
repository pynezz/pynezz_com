Code block example

22.07.2024

This is an example of a code block in markdown

package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}

This is a simple Go program that prints "Hello, world!" to the console. The code block is enclosed in triple backticks ( ` ) with the language specified after the first set of backticks.

Long horizontal code block

This is a long code block to test the horizontal scrolling.

package main

import "fmt"

func main() {
    fmt.Println("Hello, world!") // This is a comment that is long enough to test the horizontal scrolling
}

The code block should be horizontally scrollable if it exceeds the width of the container.

Inline code

You are also able to use inline code like fmt.Println("Hello, world!") in the text, which is enclosed in single backticks (```) and results in a monospaced font.

Syntax highlighting

The code blocks are syntax highlighted based on the language specified after the first set of backticks. This makes it easier to read and understand the code.

⚠️ Syntax highlighting is still a work in progress.
